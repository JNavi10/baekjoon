계단의 값을 a,b,c,d로 잡았을 때,  
d에 도착하는 방법은 다음과 같다
- a - b - d
- a - c - d

즉 d에 도착하려면 반드시 a에 도착해야한다.
d 도착을 조건 P, a 도착을 조건 Q라 한다면  P->Q 가 성립한다.

인덱스로 봤을 때 d는 3, a는 0이다.
둘다 3으로 나눈 나머지가 같다.

이것을 일반화 하여 n개의 계단이 있다고 하면 (n-1)%3 에 있는 index까지 도달하면 거기서 부터 b가 큰지 c가 큰지만 판단하면 된다.

(n-1)%3 = checkpoint라고 하고
index = 0에서 시작한다고 했을 때

checkpoint == 0:
while i != n-1
sum += stair[i] + max(stair[i+1], stair[i+2]) + ... 
i += 3
sum += stair[n-1]

checkpoint == 1:
while i != n-1:
if i == 0:
    sum += stair[0]
    i+= 1
    continue
sum += stair[i] + max(stair[i+1], stair[i+2]) + ... 
i += 3
sum += stair[n-1]

checkpoint == 2:
while i != n-1:
if i == 0:
    sum += max(stair[0] + stair[1])
    i+= 2
    continue
sum += stair[i] + max(stair[i+1], stair[i+2]) + ... 
i += 3
sum += stair[n-1]
